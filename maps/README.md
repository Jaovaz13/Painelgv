# Mapas (uso futuro)

Esta pasta destina-se a arquivos GeoJSON para visualização de mapas no painel.

- **geo_br.geojson**: geometrias do Brasil (estados ou municípios).
- **geo_mg.geojson**: geometrias de Minas Gerais (municípios).

Os arquivos podem ser obtidos em fontes como IBGE ou dados abertos. A integração com o painel (ex.: Plotly/Streamlit com choropleth) pode ser feita em uma fase posterior.
